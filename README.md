# EthioTours - Ethiopia Tourism Website

EthioTours is a captivating online platform dedicated to showcasing the breathtaking beauty and rich cultural heritage of Ethiopia. Discover ancient historical sites, diverse wildlife, vibrant traditions, and stunning landscapes through our carefully curated tours and services. Whether you're planning your next adventure or simply exploring from afar, EthioTours offers an immersive digital experience that brings the wonders of Ethiopia to your fingertips.

## ğŸš€ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Abel-berhanu-git/ethiotours-react-v1.git
   cd ethiotours-react-v1
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview the production build**
   ```bash
   npm run preview
   ```
